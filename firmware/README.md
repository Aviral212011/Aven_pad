# AVENPAD Macropad

This is a custom macropad built using XIAO RP2040. 

## Features
- 5 Mechanical Keys
- 1 Rotary Encoder
- Designed in KiCad
- case is designed in onshape

## Firmware
This repository contains a demo firmware file (code.py).

## Note
This firmware is for demonstration purposes and is not flashed to hardware.
